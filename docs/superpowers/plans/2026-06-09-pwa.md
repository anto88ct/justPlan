# PWA Setup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make AirPlan installable as a PWA on desktop and mobile with cache-first static asset caching.

**Architecture:** Install `@angular/service-worker`, wire it via `provideServiceWorker` in `app.config.ts`, configure `ngsw-config.json` with prefetch cache-first for app shell and lazy cache for other assets, add `manifest.webmanifest` for installability, generate 192/512px PNG icons from the existing `logo-mark.svg` using `sharp`.

**Tech Stack:** Angular 17 (`@angular-devkit/build-angular:application` builder), `@angular/service-worker@^17.3.0`, `sharp` (already in devDependencies), `ngsw-config.json` (Angular service worker config format).

> **Builder note:** This project uses the new `application` builder (not the legacy `browser` builder). In this builder, service worker is enabled via `"serviceWorker": "ngsw-config.json"` (a string path) — NOT the legacy `"serviceWorker": true` + `"ngswConfigPath"` pattern.

## Task 2: Generate PWA icons from SVG

**Files:**
- Create: `scripts/generate-icons.js`
- Generate: `src/assets/icons/icon-192x192.png`
- Generate: `src/assets/icons/icon-512x512.png`

- [ ] **Step 1: Create the icon generation script**

Create file `scripts/generate-icons.js` with this exact content:

```js
const sharp = require('sharp');
const path = require('path');
const fs = require('fs');

const src = path.join(__dirname, '../src/assets/logo-mark.svg');
const outDir = path.join(__dirname, '../src/assets/icons');

fs.mkdirSync(outDir, { recursive: true });

const sizes = [192, 512];

Promise.all(
  sizes.map(size =>
    sharp(src)
      .resize(size, size)
      .png()
      .toFile(path.join(outDir, `icon-${size}x${size}.png`))
  )
).then(() => {
  console.log('Icons generated:');
  sizes.forEach(s => console.log(`  src/assets/icons/icon-${s}x${s}.png`));
}).catch(err => {
  console.error('Icon generation failed:', err.message);
  process.exit(1);
});
```

- [ ] **Step 2: Run the script**

```bash
node scripts/generate-icons.js
```

Expected output:
```
Icons generated:
  src/assets/icons/icon-192x192.png
  src/assets/icons/icon-512x512.png
```

- [ ] **Step 3: Verify the files exist and are valid PNGs**

```bash
node -e "const fs=require('fs'); [192,512].forEach(s => { const f='src/assets/icons/icon-'+s+'x'+s+'.png'; const b=fs.readFileSync(f); console.log(f, b.length+'B', b[0]===0x89&&b[1]===0x50?'OK':'NOT PNG'); })"
```

Expected: both files listed with size > 1000 bytes and `OK`.

- [ ] **Step 4: Commit**

```bash

git commit -m "feat(pwa): add icon generation script and generated PNG icons"
```

## Task 7: Register service worker in app.config.ts

**Files:**
- Modify: `src/app/app.config.ts`

- [ ] **Step 1: Add provideServiceWorker to app.config.ts**

Current content of `src/app/app.config.ts`:

```ts
import { ApplicationConfig } from '@angular/core';
import { provideRouter } from '@angular/router';
import { provideAnimationsAsync } from '@angular/platform-browser/animations/async';
import { provideHttpClient } from '@angular/common/http';
import { provideTranslateService } from '@ngx-translate/core';
import { provideTranslateHttpLoader } from '@ngx-translate/http-loader';
import { routes } from './app.routes';

export const appConfig: ApplicationConfig = {
  providers: [
    provideRouter(routes),
    provideAnimationsAsync(),
    provideHttpClient(),
    provideTranslateService({ defaultLanguage: 'it' }),
    provideTranslateHttpLoader({ prefix: './assets/i18n/', suffix: '.json' }),
  ],

```

Replace with:

```ts
import { ApplicationConfig, isDevMode } from '@angular/core';
import { provideRouter } from '@angular/router';
import { provideAnimationsAsync } from '@angular/platform-browser/animations/async';
import { provideHttpClient } from '@angular/common/http';
import { provideTranslateService } from '@ngx-translate/core';
import { provideTranslateHttpLoader } from '@ngx-translate/http-loader';
import { provideServiceWorker } from '@angular/service-worker';
import { routes } from './app.routes';

export const appConfig: ApplicationConfig = {
  providers: [
    provideRouter(routes),
    provideAnimationsAsync(),
    provideHttpClient(),
    provideTranslateService({ defaultLanguage: 'it' }),
    provideTranslateHttpLoader({ prefix: './assets/i18n/', suffix: '.json' }),
    provideServiceWorker('ngsw-worker.js', {
      enabled: !isDevMode(),
      registrationStrategy: 'registerWhenStable:30000',
    }),
  ],
};
```

## Troubleshooting Reference

**`ngsw-worker.js` missing from dist:** `"serviceWorker"` in `angular.json` not set, or set to `true` instead of the string path `"ngsw-config.json"`. With the `application` builder, it must be the string.

**Manifest icons not loading:** Check `src/assets/icons/` was committed and is included in `angular.json` assets. The assets entry `"src/assets"` covers the whole `assets/` folder recursively, so no separate entry needed.

**Service worker not activating in dev:** Expected — `enabled: !isDevMode()` means SW only runs on production builds.

**`sharp` fails on Windows:** Run `npm rebuild sharp` if you hit native module errors.
